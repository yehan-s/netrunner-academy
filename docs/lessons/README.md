# NetRunner 抓包教程

> 从零开始学习网络抓包与 HTTP 调试，掌握 Reqable 工具的使用技巧

## 课程概述

本教程专为网络安全初学者和 Web 开发者设计，通过 8 节课程系统学习网络抓包技术。

**学习目标**：

- 理解 HTTP/HTTPS 协议原理
- 掌握 Reqable 抓包工具的使用
- 学会分析和调试网络请求
- 了解常见 Web 安全漏洞的发现方法

**适合人群**：

- Web 前端/后端开发者
- 移动应用开发者
- 网络安全入门者
- QA 测试工程师

如果你时间有限，推荐按以下顺序学习：

## 常见问题

### Q: 需要编程基础吗？

**A**: 不需要。本教程从零开始，适合完全没有编程经验的学习者。后续课程涉及的代码示例仅作为理解原理的辅助。

### Q: 抓包会影响网络速度吗？

**A**: 几乎不会。代理转发只增加微秒级延迟，正常使用无感知。

### Q: 学完能找安全相关工作吗？

**A**: 本教程是入门级内容，为后续深入学习打基础。如需从事安全工作，还需要学习更多专业知识（渗透测试、漏洞分析、逆向工程等）。

### Q: 可以抓取任何网站的数据吗？

**A**: 技术上可以，但请遵守法律法规。未经授权抓取他人数据可能违法。本教程仅用于学习目的，请在合法范围内使用。
